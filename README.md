# Gitmemoji

Gitmemoji is a web-based [gitmoji](https://gitmoji.dev/) quiz game. It is built using [React.js](https://reactjs.org/), [Tailwind CSS](https://tailwindcss.com/), and [Vite](https://vitejs.dev/).
